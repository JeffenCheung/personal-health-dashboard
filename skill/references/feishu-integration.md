## 集成概览

### 功能矩阵

| 模块 | 功能 | 触发方式 | 频率 |
|------|------|----------|------|
| 飞书消息 | 每日健康摘要卡片 | 生成报告后自动推送 | 每日1次 |
| 飞书消息 | 异常告警通知 | 指标异常时推送 | 按需 |
| 飞书日历 | 运动记录同步 | 检测到新运动时 | 按需 |
| 飞书日历 | 睡眠记录同步 | 检测到新睡眠数据时 | 每日1次 |
| 飞书文档 | 周报/月报同步 | 每周/每月 | 周期 |
| 飞书文档 | 仪表盘嵌入 | 生成仪表盘后更新 | 按需 |

### 认证方式

支持两种认证方式：
1. **自定义机器人 Webhook**（推荐用于消息推送）
2. **飞书开放平台应用**（用于日历、文档等高级功能）

## 飞书消息推送

### 每日健康摘要卡片

使用飞书交互式卡片（Interactive Card）展示每日健康摘要：

```
┌─────────────────────────────────┐
│  🏃 每日健康摘要 · 6月29日      │
├─────────────────────────────────┤
│  步数: 8,542 步   ↑ 12%        │
│  睡眠: 7h 32m    ↑ 5%          │
│  静息心率: 58 bpm  ↓ 3%        │
│  碎脚指数: 35  B级              │
├─────────────────────────────────┤
│  [查看完整仪表盘]  [本周报告]   │
└─────────────────────────────────┘
```

### 卡片结构
```json
{
  "msg_type": "interactive",
  "card": {
    "config": { "wide_screen_mode": true },
    "header": {
      "title": { "tag": "plain_text", "content": "🏃 每日健康摘要" },
      "template": "blue"
    },
    "elements": [
      {
        "tag": "div",
        "fields": [
          { "is_short": true, "text": { "tag": "lark_md", "content": "**步数**\n8,542 步" } },
          { "is_short": true, "text": { "tag": "lark_md", "content": "**睡眠**\n7h 32m" } }
        ]
      }
    ]
  }
}
```

### 异常告警

当检测到以下异常时推送告警：
- 静息心率异常升高（> 基线 +15%）
- HRV 显著下降（< 基线 -20%）
- 睡眠严重不足（< 5小时）
- 连续多天高碎脚指数（> 80）

告警消息样式：
```
⚠️ 健康异常提醒
┌─────────────────────────────┐
│  指标: 静息心率              │
│  当前: 72 bpm               │
│  基线: 58 bpm (+24%)        │
│  建议: 关注休息和压力水平    │
└─────────────────────────────┘
```

## 飞书日历同步

### 运动事件

将跑步/锻炼记录同步为日历事件：

- **事件标题**：🏃 夜跑 5.2km
- **事件时间**：运动开始-结束时间
- **事件描述**：
  ```
  距离: 5.2 km
  时长: 38 分钟
  平均心率: 142 bpm
  平均配速: 7'18"/km
  消耗: 420 kcal
  ```
- **日历分类**：运动（可选日历名称）

### 睡眠事件

将睡眠记录同步为日历事件：

- **事件标题**：😴 睡眠 7h32m
- **事件时间**：入睡时间 - 醒来时间
- **事件描述**：
  ```
  总睡眠: 7h 32m
  深睡: 1h 45m (23%)
  REM: 1h 30m (20%)
  核心睡眠: 4h 17m (57%)
  睡眠效率: 92%
  睡眠中点: 04:15
  ```
- **事件状态**：暂定（不标记为忙碌）

### 异常标记

在日历中标记健康异常日期：
- 用不同颜色标记：睡眠不足、运动过度、恢复不佳等
- 可选：创建"健康提醒"全天事件

## 飞书文档同步

### 健康周报/月报

将周期报告同步为飞书文档：

**文档结构**：
1. 本期概览
2. 关键指标趋势
3. 跑步分析
4. 睡眠分析
5. 心率与恢复
6. 碎脚指标分析
7. 下周/下月建议

**文档格式**：
- 使用飞书文档 Markdown 格式
- 嵌入图表截图或链接到仪表盘
- 支持评论和协作

### 仪表盘嵌入

将 HTML 仪表盘嵌入飞书文档：
- 使用飞书文档的"网页嵌入"功能
- 或上传 HTML 文件作为附件
- 或生成仪表盘截图插入文档

## 配置说明

### 配置文件格式

```json
{
  "feishu": {
    "enabled": true,
    "webhook": {
      "daily_summary": "https://open.feishu.cn/open-apis/bot/v2/hook/xxx",
      "alert": "https://open.feishu.cn/open-apis/bot/v2/hook/yyy"
    },
    "app": {
      "app_id": "cli_xxx",
      "app_secret": "xxx",
      "user_access_token": "xxx"
    },
    "calendar": {
      "enabled": true,
      "calendar_id": "primary",
      "sync_workouts": true,
      "sync_sleep": true
    },
    "doc": {
      "enabled": true,
      "weekly_report_folder": "fldcnxxx",
      "dashboard_doc_token": "doccnxxx"
    },
    "notify": {
      "daily_summary": true,
      "alert_on_abnormal": true,
      "alert_thresholds": {
        "resting_hr_increase": 0.15,
        "hrv_decrease": 0.20,
        "min_sleep_hours": 5,
        "max_fragmentation": 80
      }
    }
  }
}
```

### 获取配置参数

#### Webhook URL
1. 飞书群 → 设置 → 群机器人 → 添加机器人 → 自定义机器人
2. 复制 Webhook 地址
3. 可选：设置签名密钥提高安全性

#### 应用凭证（高级功能）
1. 访问飞书开放平台：https://open.feishu.cn/
2. 创建企业自建应用
3. 获取 App ID 和 App Secret
4. 开通所需权限：日历读写、文档读写等

## 错误处理

### 重试策略

| 错误类型 | 重试策略 | 最大重试 |
|----------|----------|----------|
| 网络超时 | 指数退避重试 | 3次 |
| 429 限流 | 等待后重试（按 Retry-After） | 5次 |
| 401 认证失败 | 刷新 token 后重试 | 1次 |
| 4xx 其他错误 | 不重试，记录日志 | 0次 |
| 5xx 服务器错误 | 等待后重试 | 3次 |

### 降级策略

1. **消息推送失败**：记录日志，不影响其他功能
2. **日历同步失败**：保存到本地队列，下次重试
3. **文档同步失败**：生成本地文件，手动上传

### 日志记录

每次飞书同步操作记录日志：
- 时间戳
- 操作类型
- 成功/失败状态
- 错误信息（如失败）
- 数据摘要
